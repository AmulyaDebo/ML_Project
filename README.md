STAT_ML PROJECT SUBMISSION README: 
 
SEQUENCE OF EXECUTION: 
 
1.  statML_Preprocessing.ipynb: 
It consists of the pre-processing steps.  
•  Feature Reduction 
•  Removes Duplicate Rows 
•  Removes N/A 
•  Imputes the inf values

2.  statML_lr.ipynb: 
It consists of Logistic Regression algorithm. Both weighted and non-weighted. 
3.  statML_knn.ipynb: 
It consists of K Nearest Neighbours algorithm 
4.  statML_stack_XGBOOST.ipynb: 
It consists of XGBOOST algorithm and weighted logistic regression. Using these two, 
a StackClassifier is also present. 
5.  statML_nn: 
It consists of Neural Networks using Keras
